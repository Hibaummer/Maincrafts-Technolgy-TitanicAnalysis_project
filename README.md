# Titanic Dataset Analysis
## 📌 Project Overview
#### This project is part of the Data Science & Data Analysis with Python Internship – Task 2.The goal is to analyze the famous Titanic dataset and uncover meaningful insights about passenger survival patterns using data analysis and visualization techniques.The dataset includes passenger details such as age, gender, class, and survival status. Through this project, we explore how these factors influenced survival outcomes.
## 🎯 Objectives
#### 1.Load and explore the Titanic dataset
#### 2.Perform data cleaning and preprocessing
#### 3.Analyze survival patterns based on different features
#### 4.Visualize insights using graphs and charts
## 📂 Dataset
#### Source: Kaggle Titanic Dataset
#### Features include:
##### PassengerId
##### Pclass (Passenger Class)
##### Name
##### Sex
##### Age
##### SibSp (Siblings/Spouses aboard)
##### Parch (Parents/Children aboard)
##### Fare
##### Embarked
##### Survived (Target Variable)
